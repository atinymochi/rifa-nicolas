# 🎫 Rifa Solidaria — Sistema de Gestión en Tiempo Real

Este es un sistema web minimalista y responsivo para administrar la venta de una rifa de **300 números**. La aplicación permite a los usuarios visualizar la disponibilidad de los números en tiempo real y ofrece un panel de administración integrado para gestionar las ventas sin necesidad de recargar la página.

## ✨ Características

- **Visualización en Tiempo Real:** Sincronización instantánea de los números vendidos y disponibles mediante **Firebase Realtime Database**.
- **Modo Administrador Integrado:** Acceso seguro mediante contraseña local para gestionar la rifa desde la misma pestaña.
- **Gestión Avanzada de Números:** - Asignación rápida de números a compradores.
  - Edición directa del nombre del dueño de un número ya vendido.
  - Liberación instantánea de números (cancelación de venta).
- **Interfaz Fluida (UX):** Soporte para atajos de teclado (`Enter` para confirmar, `Esc` para cerrar), notificaciones flotantes (Toasts) y Tooltips interactivos para ver los nombres de los compradores.
- **Diseño Dark / Cyberpunk:** Estética moderna optimizada para dispositivos móviles y computadoras de escritorio.

## 🛠️ Tecnologías Utilizadas

- **Frontend:** HTML5, CSS3 (Variables nativas, CSS Grid, Flexbox), JavaScript Moderno (ES6+ Modules).
- **Backend / Base de Datos:** [Firebase Realtime Database](https://firebase.google.com/) (Web SDK v12).
- **Tipografías:** *Syne* y *DM Mono* via Google Fonts.
